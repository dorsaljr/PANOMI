# PANOMI UI Design Reference

## Current Version: 1.4.1

A unified game launcher that detects and organizes all your games and launchers in one place.

---

## Colors

### Backgrounds
- **App background:** `#0f1114` (near black)
- **Cards:** `#161a1e`
- **Card hover:** `#1e2328`
- **Stats boxes:** `#0f1114`

### Text
- **Primary text:** `#FFFFFF` (white)
- **Secondary/Label text:** `#888888` (gray)

### Accent Colors
- **Primary accent:** `#00d4ff` (cyan)
- **Favorite star:** `#facc15` (yellow)
- **Update button:** `#4ade80` (green)

---

## Layout

### Library Cards (Compact Style)
- **Height:** 56px horizontal bars
- **Min width:** 300px
- **Layout:** Icon (32x32) | Name | Play (▶) | Favorite (☆)
- **Background:** `#161a1e`, hover `#1e2328`
- **Tooltip:** Shows full name only when truncated

### Stats Boxes
- Total, Launchers, Games
- Large number (~32px bold), small label above
- Dark background, rounded corners

## Features

### Supported Launchers (10)
Steam, Epic Games, GOG Galaxy, EA App, Ubisoft Connect, Battle.net, Riot Games, Rockstar Games, Minecraft, Roblox

### Localization (29 Languages)
English, Español, Português, Français, Deutsch, Italiano, 한국어, 日本語, 中文, हिन्दी, Русский, Polski, Türkçe, العربية, اردو, Українська, Indonesia, Tiếng Việt, ไทย, فارسی, বাংলা, Kiswahili, Ελληνικά, Română, Čeština, Српски, Nederlands, Shqip, Tagalog

### Quick Launch
- Minimize to system tray for instant access
- Configurable in settings

### Web Menu
- Console, Live Streaming, Misc (itch.io, GitHub, etc.), PC, Social Media, TV Streaming, Stores
- Chinese Platforms (zh-CN only): Baidu, Bilibili, Douyin, iQIYI, Mango TV, Tencent Video, WeChat, Weibo, Xiaohongshu, Youku

### Auto-Updates
- Velopack integration
- Localized "Update Available" button

---

## Design Principles

1. **Compact layout** - More games visible at once
2. **Minimal UI** - Play icon (▶) instead of "Launch" text
3. **Dark theme** - Gaming aesthetic
4. **Privacy first** - No telemetry, no accounts, local only
5. **Signed & Certified** - Microsoft Azure Trusted Signing
