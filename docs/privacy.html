Privacy Policy

Last updated: January 12, 2026

Overview

Panomi is a unified game launcher that helps you organize and launch games from multiple platforms. We are committed to protecting your privacy.

Data Collection

Panomi does not collect, transmit, or store any personal data on external servers.

All data is stored locally on your device, including:

  • Game library information (detected from installed launchers)
  • Application settings and preferences
  • Game icons and metadata

Data Storage

All application data is stored in your local AppData folder:

%LOCALAPPDATA%\Panomi

You can delete this folder at any time to remove all Panomi data from your device.

Third-Party Services

Panomi reads game information from launchers you have installed (Steam, Epic Games, GOG, etc.) but does not communicate with these services directly. Panomi only reads locally stored data.

Network Access

Panomi may access the internet only for:

  • Checking for application updates (GitHub releases version only)
  • Opening web links when you click on them in the app

No personal data is transmitted during these operations.

Children's Privacy

Panomi does not collect any personal information from anyone, including children under 13 years of age.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
